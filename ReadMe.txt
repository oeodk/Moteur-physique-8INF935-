Comment exécuter le projet :

1. Créer un projet avec OpenFrameworks.
2. Remplacer les fichiers du dossier src du projet nouvellement créé par les fichiers src de notre projet.
3. Remplacer les éléments du dossier bin/data par ceux de notre projet.
4. Inclure les fichiers source dans Visual Studio (n'importe laquelle des méthodes citées):
   -Clic droit dans l'explorateur de solutions sur les fichiers, puis "Inclure dans le projet".
   -Clic droit sur "Source", puis ajouter les éléments du dossier.
5. Exécuter le code avec Visual Studio.

Fonctionnement du programme :
 - Il est possible de se déplacer avec ZQSD ou WASD
 - La touche Ctrl gauche permet de sprinter
 - Les touches Espace et Maj gauche permettent de monter et descendre
 - Il est possible de bouger la caméra avec la souris, cliquer permet de bloquer ce mouvement
 - La molette de la souris permet de changer de particule
 - Le clique gauche de la souris permet de tirer une particule
